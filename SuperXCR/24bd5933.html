int 与long 的区别

int

最小值 : Integer.MIN_VALUE=-2147483648 (-2的31次方)

最大值 : Integer.MIN_VALUE=2147483647 (2的31次方-1)

long

最小值 : Long.MIN_VALUE=-2147483648 (-2的63次方)

最大值 : Long.MIN_VALUE=2147483647 (2的63次方-1)

注意long类型赋值的时候要在数字最后跟上一个小写的 l (是字母L)
